# 🚢 Titanic Survival Predictor

A machine learning web app that predicts whether a passenger would have survived the Titanic disaster.

# Project Overview
- Performed exploratory data analysis to find survival patterns
- Discovered that gender and fare were the strongest survival factors
- Trained and compared two models — Logistic Regression (81%) and Random Forest (82%)
- Built an interactive web app using Streamlit

# Technologies Used
- Python
- Pandas, NumPy, Matplotlib, Seaborn
- Scikit-learn
- Streamlit

# How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
